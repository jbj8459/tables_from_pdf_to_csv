# tables_from_pdf_to_csv

## Feature
camelot으로 pdf의 table을 읽어서 csv로 만듭니다.
한 개의 pdf 파일만 됩니다.

## Usage
**--pdf, -p**    파일 경로

**--output, -o**    출력 폴더 경로

**--start, -s**    시작 페이지

**--end, -e**    마지막 페이지

**--style, -st**   테이블 모양 (stream: 줄이 없는 테이블 / lattice: 줄이 있는 테이블)


## Trivia
stream 형식은 쓸 데 없는 내용도 출력됩니다...
